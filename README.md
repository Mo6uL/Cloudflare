# Cloudflare

Jednoduchá webová prezentace o technologii Cloudflare vytvořená v rámci školního projektu.

## Živý web
Projekt je publikován pomocí GitHub Pages na adrese:
https://mo6ul.github.io/Cloudflare/

## Obsah a struktura
* **index.html** – Obsahuje stručné vysvětlení platformy a popis 3 hlavních služeb (CDN, DDoS ochrana, DNS).
* **style.css** – Vlastní moderní stylování v tmavých a fialových tónech s responzivním dvou sloupcovým rozvržením.
